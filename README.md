# Anderson Gonçalves | Personal Trainer Landing Page

Professional landing page developed for **Anderson Gonçalves**, a personal trainer and online mentor. This project was built with a modern tech stack to ensure performance, maintainability, and scalability. Fully responsive, visually engaging, and optimized for production deployment.

---

## 💡 Project Overview

This is a real-world freelance project aimed at delivering a complete web solution for a personal training business. It includes a custom layout, form submission via email using FastAPI, and deployment workflows leveraging modern DevOps practices.

---

## 🔧 Tech Stack

**Frontend**  
- Next.js 15 (App Router)
- TypeScript
- Styled Components
- Responsive Layout
- Vercel Hosting

**Backend**  
- FastAPI (Python)
- Email-based form handling (no database)
- Render or AWS Deployment

**DevOps**  
- Docker
- GitHub Actions (CI/CD)
- Environment-based builds
